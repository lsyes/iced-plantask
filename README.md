# iced-plantask

一个基于[Iced](https://iced.rs/) GUI框架开发的跨平台简易系统计划任务管理器。

## 目前已经做的功能

- **定时关机** - 延时关机
- **立即关机** - 立即关机
- **取消任务** - 取消关机任务

## 使用方法
1. 在输入框中设置分钟数（默认为40分钟）
2. 点击相应按钮执行操作：
   - **立即关机** - 马上执行关机
   - **延时 X 分钟** - 在指定时间后关机
   - **取消任务** - 取消已设置的关机任务
   - **关于** - 打开关于窗口

## 运行本程序

cargo run
